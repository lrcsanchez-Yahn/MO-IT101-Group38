## Program Details

The **MotorPH Payroll System** is a Java-based application that calculates employee payroll using data from employee and attendance CSV files.

### Key Features
- **Login Authentication:** Users can log in as either an employee or payroll staff.  
- **Employee Access:** Employees can view their personal information.  
- **Payroll Staff Access:** Payroll staff can process payroll for individual employees or for all employees.  
- **Attendance & Hours Calculation:** The system computes total hours worked based on attendance records.  
- **Payroll Computation:** Calculates gross salary and applies government-mandated deductions including **SSS, PhilHealth, Pag-IBIG, and Tax**.  
- **Payroll Display:** Results are displayed for each cutoff period from **June to December**, ensuring accurate monthly payroll reporting.
